## 🚀 Overview

Whitebox AI is a full-stack AI chatbot built using modern Large Language Models (LLMs).  
It provides real-time intelligent responses with a clean and responsive interface.

This project demonstrates:
- AI model integration (Google Gemini API)
- Real-time chat system
- Backend API handling
- Prompt-based interaction

---

## ✨ Features

- 💬 Real-time conversational AI  
- 💻 Code generation & debugging  
- 🔢 Math problem solving  
- 🖼️ Image-based queries  
- 📁 File upload & analysis  
- 💾 Chat history (local storage)  
- 🌙 Dark mode UI  
- ⚙️ Custom settings
